Apasih Python itu?
    Python adalah salah satu dari banyaknya bahasa pemrograman yang tersebar di seluruh dunia dan sering menjadi 5 bahkan 3 besar bahasa pemrograman yang dibutuhkan saat ini.
    Python ngga hanya buat website aja lo. Tapi masih banyak kegunaan yang lain, mulai dari Pengembangan IOT, Machine Learning, Data Science, dan masih banyak lagi yang berkaitan dengan bahasa Python ini.

Kelebihan Python itu apa sih?
    - Gratis
    - Python itu mudah dipelajari dan sangat direkomendasikan kepada anda yang baru belajar bahasa pemrograman dengan Syntax yang simpel.
    - Python support dengan multi platform sehingga para programmer lebih mudah dalam kolaborasi.
    - Python bersifat Object Oriented Programming.
    - Menghemat waktu bagi programmer karena syntax yang sangat simpel.

Kekurangan Python itu apa sih?
    - Python bisa disebut lemah dalam proyek pembuatan aplikasi platform mobile.
    - minimnya dukungan multiprosesor sehingga developer agak kesulitan karena terbatas dalam penulisan kode.
    - Kecepatan dari Python ini yang sering diprotes prosesnya yang lama daripada bahasa pemrograman lain.

Penerapan dan prospek kerjanya gimana nih untuk bahasa python ini?
    Python ini sifatnya fleksibel dan membuatnya bisa digunakan di berbagai industri seperti Manufaktur, Pendidikan, Kesehatan, Manufaktur , Teknologi, dan masih banyak lagi.
    Dengan menguasai Python, kamu bisa berkarir denganberbagai macam posisi dan kegunaan, seperti software Desktop, Website, Games, IOT, dan masih banyak lagi.
